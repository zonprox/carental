## Visual Differentiation

### Color Schemes
- **UserNavMenu**: Neutral blue theme (bg-muted/50, ring-border)
- **Admin UserMenu**: Orange admin theme (bg-orange-50, ring-orange-200)

### Menu Structure
- **UserNavMenu**: 9-10 items focused on user experience and rentals
- **Admin UserMenu**: 13 items focused on system management

### Role-Based Features
- Admin users see "Trang quản trị" link in UserNavMenu (top position)
